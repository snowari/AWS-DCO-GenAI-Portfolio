# 교육용 샘플 데이터 사용

# SAMPLE DCO Ticket

## 1. 티켓 ID

EDU-TICKET-SAMPLE-CRC-LINKDOWN

## 2. 발생 시간

EDU-YYYY-MM-DD HH:MM

## 3. 샘플 장비명

SAMPLE_TOR_SW_01

## 4. 이벤트

- CRC error 증가 관찰
- Link Down 이벤트 관찰

## 5. 심각도

EDU-MEDIUM

## 6. 관찰 내용

교육용 샘플 로그에서 `SAMPLE_TOR_SW_01`과 관련된 CRC error 증가 표현이 먼저 관찰되었습니다. 이후 같은 샘플 장비명에서 Link Down 이벤트가 함께 표시된 것으로 정리합니다.

이 문서는 실제 장비 점검 결과가 아니라, 샘플 로그를 읽고 발생 시간, 이벤트 종류, 반복 여부, 전달 필요성을 기록하는 연습용 티켓입니다.

## 7. Escalation 필요 여부

필요 여부: EDU-확인 필요

사유: CRC error 증가와 Link Down 이벤트가 같은 샘플 장비명에서 함께 관찰되었으므로, 원인을 단정하지 않고 담당자에게 전달할 수 있도록 Escalation 필요 여부를 표시합니다.
