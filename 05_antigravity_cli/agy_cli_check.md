  ──────
  ### 📂 폴더 구조 및 역할 요약

   폴더명               | 역할 및 학습 내용
  ----------------------|-----------------------------------------------------------------------------------------------------------------------------------
   📁 01_dco_glossary    | DCO 핵심 용어 정리: DCO 직무와 데이터센터 인프라 관련 핵심 용어를 정리하는 공간입니다.
   📁 02_dco_profile    | 개인 역량 프로필: 협업 유형, 강점 진단 등 DCO 직무와 연결되는 개인의 강점과 역량을 정리합니다.
   📁 03_sop_analysis    | SOP 분석 및 체크리스트: 교육용 SOP(표준운영절차)를 분석하여 목적, 절차, 주의사항, 확인 항목을 구조화하고 체크리스트로 변환합니다.
   📁 04_environment_setup    | 개발/실습 환경 구축: Git, GitHub, VS Code, Python, Node.js 등 로컬 실습 환경의 설정 결과를 기록합니다.
   📁 05_antigravity_cli    | Antigravity CLI 실습: 개발/운영 보조 인공지능 도구인 Antigravity CLI의 실습 산출물을 정리합니다.
   📁 06_cli_file_automation    | CLI 파일 자동화: CLI를 활용한 파일 생성, 이동 및 자동화 스크립트 작성 실습 내용을 정리합니다.
   📁 07_log_analysis_script    | 로그 분석 스크립트: 교육용 인프라 로그를 분석하기 위한 Python 스크립트 및 실행 결과를 정리합니다.
   📁 08_incident_analysis    | 장애 원인 분석: 가상의 장애 상황을 정의하고, 근본 원인(Root Cause) 후보와 점검 절차를 설계합니다.
   📁 09_incident_report    | 장애 대응 보고서: 분석한 장애 상황에 대한 보고서 초안을 작성하고 피드백을 통해 개선한 최종 보고서를 정리합니다.
   📁 10_stitch_dco_dashboard   | DCO 대시보드 구축: 지금까지의 학습 산출물과 모니터링 데이터를 시각화하는 대시보드 실습 결과를 정리합니다.
   📁 11_exam_prep   | 시험 대비 요약: 지필시험 대비를 위해 DCO 핵심 이론, 확인 절차, 서술형 예시 답안 등을 종합 정리합니다.

## 8. 보안 확인 메모

- 실제 내부 SOP, 실제 티켓 번호, 실제 장비명, 실제 IP 주소, 계정정보, API 키는 프롬프트에 입력하지 않습니다.
- 수업 실습에서는 교육용 샘플 데이터와 공개 직무 이해 맥락만 사용합니다.
- AI 응답은 초안으로 활용하고, 실제 내부 절차처럼 보이는 표현이나 원인을 단정하는 표현은 직접 검토합니다.
