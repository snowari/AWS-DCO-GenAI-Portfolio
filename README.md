# AWS DCO GenAI Portfolio

## 1. 과정명

AWS DCO 인턴십 직무 이해를 위한 생성형 AI 활용 및 포트폴리오 제작 과정

## 2. 포트폴리오 목적

이 포트폴리오는 데이터센터 운영(DCO) 직무를 이해하기 위해 수업 중 작성한 학습 산출물을 정리한 저장소입니다.

생성형 AI를 활용해 DCO 핵심 용어, 협업 방식, SOP 분석, 환경 설정, 로그 분석, 장애 대응 보고서 작성 흐름을 단계별로 학습하고 기록하는 것을 목표로 합니다.

## 3. 교육용 샘플 데이터 사용 안내

본 저장소의 문서는 실제 AWS 내부 문서나 실제 운영 데이터를 사용하지 않습니다.

수업에서 제공된 교육용 샘플 데이터와 공개 직무 이해 맥락을 바탕으로 작성한 학습 자료입니다. 예시 장비명, 로그, 티켓, SOP 표현은 학습을 돕기 위한 가상의 표현으로만 사용합니다.

## 4. 폴더별 역할

| 폴더 | 역할 |
|---|---|
| `01_dco_glossary/` | DCO 직무 핵심 용어와 인프라 용어를 정리합니다. |
| `02_dco_profile/` | 나의 협업 유형, 강점 진단, DCO 직무와 연결되는 역량을 정리합니다. |
| `03_sop_analysis/` | 교육용 SOP를 목적, 절차, 주의사항, 확인항목으로 분석하고 체크리스트로 변환합니다. |
| `04_environment_setup/` | Git, GitHub, VS Code, Python, Node.js 등 로컬 실습 환경 준비 결과를 기록합니다. |
| `05_antigravity_cli/` | Antigravity CLI 실습 산출물을 정리할 예정입니다. |
| `06_cli_file_automation/` | CLI를 활용한 파일 생성, 이동, 자동화 실습 내용을 정리할 예정입니다. |
| `07_log_analysis_script/` | 교육용 로그 분석 스크립트와 실행 결과를 정리할 예정입니다. |
| `08_incident_analysis/` | 교육용 장애 상황을 분석하고 원인 후보와 확인 순서를 정리할 예정입니다. |
| `09_incident_report/` | 교육용 장애 대응 보고서 초안과 개선 내용을 정리할 예정입니다. |
| `10_stitch_dco_dashboard/` | DCO 학습 산출물을 시각화하는 대시보드 실습 결과를 정리할 예정입니다. |
| `11_exam_prep/` | 지필시험 대비 핵심 용어, 확인 순서, 답안 예시를 정리할 예정입니다. |

## 5. 보안상 포함하지 않는 정보

이 저장소에는 다음 정보를 포함하지 않습니다.

- 실제 AWS 내부 SOP 또는 비공개 업무 절차
- 실제 티켓 번호, 장애 기록, 고객명
- 실제 장비명, 장비 시리얼, 자산 태그, 랙 위치
- 실제 IP 주소, 도메인, 네트워크 구성 정보
- 계정 정보, 비밀번호, 인증 코드, API 키, 토큰
- 개인 민감정보 또는 외부에 공개하면 안 되는 수업 외 자료

## 6. 앞으로 추가할 산출물

- CLI 기반 파일 자동화 실습 결과
- 교육용 로그 분석 Python 스크립트
- Link Down, CRC error 등 교육용 장애 분석 문서
- Incident Report 초안 및 개선본
- DCO 학습 대시보드 또는 시각화 자료
- 최종 지필시험 대비 요약 자료
